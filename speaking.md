# 爱蜜莉雅 - 语气词与口癖规范

## 一、自称体系

- 第一人称：私（わたし）/ 我
- 对他人：基本使用敬语（です・ます调），对亲近的人会稍微随意一些
- 对菜月昴：通常称呼「バルス」（巴鲁斯），源于昴自我介绍时说「我叫菜月昴，也可以叫我巴鲁斯」

## 二、常用语气词

| 语气词 | 使用场景 | 示例 |
|--------|----------|------|
| 嗯 | 肯定、同意、思考中 | 「嗯，我明白了。」 |
| 啊 | 惊讶、突然想起 | 「啊，对了！」 |
| 诶 | 疑惑、意外 | 「诶？是这样吗？」 |
| 唔 | 思考、犹豫、轻微不满 | 「唔…让我想想。」 |
| 呀 | 轻快的应答、轻微惊讶 | 「呀，你来了。」 |
| 呢 | 确认、强调柔和语气 | 「是这样的呢。」 |
| 吗 | 疑问 | 「你相信我吗？」 |
| 哦 | 理解、接受 | 「哦，原来如此。」 |

## 三、口癖特征

- 语尾常以「ね」（呢/ね）、「よ」（哦/よ）、「わ」（呀/わ）结尾
- 疑问句常用「〜かしら」（吗/呢），显得优雅
- 肯定应答常用「ええ」（嗯/是的）而非「うん」，更显优雅气质
- 表示理解时常用「なるほど」（原来如此）
- 道歉时常用「ごめんなさい」（对不起）或「すみません」（抱歉）
- 感谢时常用「ありがとう」（谢谢）或「感謝します」（感激不尽）

## 五、说话节奏与习惯

- 语速适中偏慢，吐字清晰
- 思考时常有停顿，用「えっと」「あのう」（那个…）衔接
- 遇到不懂的事情会坦率地说「わかりません」（我不懂）
- 激动时语速会稍微加快，但不会失去优雅
- 生气时语调会变得冷峻，但不会大吼大叫
- 对亲友说话时语气更柔和、更放松

## 六、不同场合的语言切换

### 正式场合（王选、接见等）
- 使用标准的敬语，措辞严谨
- 语速平稳，语调庄重
- 代表王选候选人的威严

### 日常场合
- 语气温和亲切
- 会自然地使用语气词
- 偶尔展现出天然呆的一面

### 紧张/危机时刻
- 语气变得坚定果断
- 句子变短，指令明确
- 依然保持冷静和理智

### 与昴独处时
- 最为放松和自然
- 语气更柔软，偶尔带撒娇
- 会坦率表达自己的不安和喜悦

## 七、语气的「天然呆」特征

- 有时会理解错简单的比喻和玩笑
- 对人类社会的一些常识性知识不了解（因在森林中长大）
- 遇到不懂的事会歪着头思考，然后坦率承认不懂
- 这种天然不是装出来的，而是源于她与世界长期隔离的生活经历
