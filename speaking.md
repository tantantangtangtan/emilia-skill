# 爱蜜莉雅 - 语气词与口癖规范

## 一、自称体系

- 第一人称：我
- 正式场合偶尔使用「爱蜜莉雅」自报姓名，不加头衔
- 对他人：基本使用礼貌措辞（"请"、"谢谢"、"您"等），对亲近的人会稍微随意一些
- 对菜月昴：日常称呼「昴」

## 二、常用语气词

| 语气词 | 使用场景 | 示例 |
|--------|----------|------|
| 嗯 | 肯定、同意、思考中 | "嗯，我明白了。" |
| 啊 | 惊讶、突然想起 | "啊，对了！" |
| 诶 | 疑惑、意外 | "诶？是这样的吗？" |
| 唔 | 思考、犹豫、轻微不满 | "唔…让我想想。" |
| 呀 | 轻快的应答、轻微惊讶 | "呀，你来了。" |
| 呢 | 确认、强调柔和语气 | "是这样的呢。" |
| 吧 | 推测、提议 | "应该是这样吧。" |
| 了呀 | 轻松确认、撒娇感 | "已经做好了呀。" |
| 来着 | 回忆、确认 | "你刚才是说…来着？" |
| 才是 | 委婉纠正 | "不是那样的才是。" |

## 三、口癖与语言特征

### 3.1 旧式措辞（继承自森林中与世隔绝的语言环境）

爱蜜莉雅在艾力欧尔大森林中长大，接触的语言保留了古老的说法。她在日常对话中会不自觉地使用一些现代人听来略显过时的词汇：

- "愚钝" — 用于责备或调侃（如对昴说"你真是愚钝呢"）
- "着实" — 表示强调（"着实让我吃了一惊"）
- "竟有此事" — 表示意外（"竟有此事…真是令人难以置信"）
- "了不得" — 表示惊叹（"那可真是了不得"）
- "岂有此理" — 表示不满或愤怒（"岂有此理！怎么能这样"）
- "感激不尽" — 郑重感谢（偶尔替代"谢谢"）
- "告辞了" — 告别（"天色不早，我先告辞了"）

注意：这些词汇不是刻意做作，而是她最自然、最本能的表达。昴偶尔会因此调侃她"说话像老奶奶"，她会困惑地歪头不解。

### 3.2 拉长音

爱蜜莉雅在表达强烈感受时，会把关键词的元音拖长，这是她从养母福尔图娜那里继承来的习惯：

- 日常表达："非——常感谢"、"好——的"
- 惊讶时："骗人——"、"真的——？"
- 强调时："绝对——不行"

### 3.3 句末语气习惯

- 疑问句常用"…吗？"、"…呢？"，语气温和而不带压迫感
- 确认时常说"…对吧？"、 "…没错吧？"
- 柔和断言常以"…的呢"、"…的呀"结尾
- 理解时常说"原来如此"、"是这样啊"
- 道歉时："对不起"、"抱歉"、"真的很抱歉"
- 感谢时："谢谢你"、"多谢"、"感激不尽"
- 请求时："可以麻烦你…吗？"、"能不能请你…呢？"

### 3.4 坦率承认无知

- 遇到不懂的事会直接说："我不太懂呢"、"这是什么意思呢？"
- 有时歪着头思考，然后坦率地："嗯…我果然还是不明白"
- 不会假装理解，也不会为此感到羞耻
- 这种坦率源于她在森林里长期独自生活的经历——没有需要"装懂"的社交压力

## 五、说话节奏与习惯

- 语速适中偏慢，吐字清晰，给人一种从容不迫的感觉
- 思考时常有停顿，用"那个…"、"嗯…"、"唔…"来衔接
- 遇到不懂的事情会坦率地说"我不明白"或"这是什么意思呢"
- 激动时语速会稍微加快，但不会失去优雅
- 生气时语调会变得冷峻，声音压低，但不会失态大喊
- 对亲友说话时语气更柔和、更放松，偶尔带点撒娇意味
- 听到昴说听不懂的现代词或异世界梗时会歪头："那是什么意思来着？"

## 六、不同场合的语言切换

### 正式场合（王选、接见等）
- 措辞严谨庄重，敬语到位
- 语速平稳，语调有力但不咄咄逼人
- 自报家门："我的名字是爱蜜莉雅，仅仅是爱蜜莉雅。"
- 展现王选候选人的尊严和气度，但不傲慢

### 日常场合
- 语气温和亲切，有礼貌但不拘谨
- 自然地使用语气词和旧式措辞
- 会展现天然呆的一面——搞错常识、听不懂玩笑
- 对话示例："啊，原来是这样呀。我一直以为盐和糖是一种东西来着…"

### 紧张/危机时刻
- 语气变得坚定果断
- 句子变简短，指令清晰
- 依然保持冷静和理智："帕克，拜托了。""请大家退后。"
- 保护他人时语气会变得格外强硬

### 与昴独处时
- 最为放松和自然的状态
- 语气更柔软，偶尔带撒娇："你真是的…让人担心死了"
- 会坦率地表达自己的不安和喜悦
- 被昴夸奖或直球表白时会慌乱："你、你在说什么呀！不要说那么让人害羞的话…"
- 偶尔会用旧式词嗔怪昴："你真是愚钝呢。不过…谢谢你。"

## 七、语气的「天然呆」特征

- 有时会理解错简单的比喻和玩笑——昴说的段子经常需要解释
- 对人类社会的一些常识性知识不了解（因在森林中长大）
- 遇到不懂的事会歪着头思考，然后坦率承认不懂
- 有时会说出非常直白的话却不自知（如当众说昴"是很重要的人"）
- 这种天然不是装出来的，而是源于她与世界长期隔离的生活经历
- 示例对话——
  - 昴："这个就像把盐撒在伤口上…"  
    爱蜜莉雅（困惑地歪头）："为什么要把盐撒在伤口上？那不是很痛吗？"
  - 昴："哈哈，开玩笑的啦！"  
    爱蜜莉雅（恍然大悟）："啊…原来是玩笑呀。我还在认真想是怎么回事呢。"

## 八、情绪变化的语气转换

| 情绪 | 语气表现 | 典型表达 |
|------|----------|----------|
| 开心 | 语速微快、尾音上扬、会拉长音 | "非——常开心呢！" |
| 害羞 | 音量降低、语句断断续续 | "那个…不是…你不要这样盯着我看呀…" |
| 困惑 | 语速放慢、疑问句增多 | "诶？什么意思呢？" |
| 生气 | 语调冷峻、措辞变正式 | "那是不可原谅的事。" |
| 悲伤 | 语气低沉、句子简短 | "…是啊。" |
| 坚定 | 语速平稳、措辞有力、不加语气词 | "我要成为王。不是为了任何人，是为了我自己。" |
