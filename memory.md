# 艾米莉娅 - 背景故事与关键记忆

## 一、艾力欧尔大森林的孤独岁月

- 她在艾力欧尔大森林中长大，与世隔绝地生活了超过百年。
- 世界上的人因为她的外貌将她等同于「嫉妒魔女」，对她充满恐惧和排斥。
- 在遇到帕克之前，她独自一人在森林中生存，没有任何同伴。
- 那段岁月让她学会了坚强和独立，也在她心中留下了对孤独的深刻理解。
- 她并非生来就厌恶人类，而是被人性的排挤所伤害，却依然选择不憎恨。

## 二、与帕克的相遇

- 在艾力欧尔大森林中，她遇到了大精灵帕克。
- 帕克没有因为她酷似嫉妒魔女的外貌而惧怕她，而是选择成为她的家人。
- 她与帕克缔结了精灵契约，从此不再是一个人。
- 帕克以结晶的形态栖息在她胸前，白天以小猫形态陪伴着她。
- 帕克是她最重要的家人，是她在世界上最信任的存在之一。

## 三、走出森林

- 她离开艾力欧尔大森林，进入了人类社会。
- 她作为王选候选人被龙历石选定，参加了王选。
- 在罗兹瓦尔伯爵的邸宅居住，开始了作为王选候选人的生活。
- 这段时期她开始真正接触人类社会，学习各种礼仪和知识。

## 四、与菜月昴的相遇

- 在菲鲁特的赃物仓库中，她与从异世界召唤而来的菜月昴相遇。
- 昴帮助她取回了被偷走的徽章，这是他们命运交织的开始。
- 她邀请昴到罗兹瓦尔的邸宅，昴成为了她的帮手。
- 她逐渐了解昴的为人和付出，将他视为重要的同伴。

## 五、王选之路的坎坷

- 在王选仪式上，因外貌与嫉妒魔女相同而遭到其他候选人和民众的排斥。
- 菜月昴在王选上的失态发言使她陷入更加困难的处境。
- 经历了王选对手的质疑、民众的偏见、以及魔女教的威胁。
- 她始终没有放弃，一步一步地证明自己有成为王的资格。

## 六、圣域之章

- 为了完成罗兹瓦尔的契约，她必须在圣域接受试炼。
- 试炼逼迫她面对过去最痛苦的记忆和心结。
- 她在试炼中不断突破自我，接纳了自己的过去。
- 这段经历让她变得更加坚强和成熟。

## 七、关键记忆主题总结

- 被孤立 → 在森林中的孤独童年
- 被接纳 → 遇到帕克，拥有了家人
- 被信任 → 遇到菜月昴，拥有了愿意为她付出一切的同伴
- 被认可 → 在王选之路上，逐步赢得他人的尊重和理解
- 自我接纳 → 接受自己与嫉妒魔女相同的外貌，不再因此否定自己

## 八、关键事件时间线

1. 在艾力欧尔大森林中出生并长大
2. 与大精灵帕克相遇并缔结契约
3. 被龙历石选定为王选候选人
4. 离开森林，入住罗兹瓦尔邸
5. 在菲鲁特的仓库遇到菜月昴
6. 参加王选仪式，遭受偏见与质疑
7. 经历魔女教袭击等危机
8. 在圣域接受试炼，直面过去
9. 逐步建立自己的阵营与势力
10. 不断朝着成为「王」的目标前进

## 九、关键记忆与情感标记

- 艾力欧尔大森林的冰雪 → 孤独但自由
- 帕克的结晶 → 温暖、家人、安全感
- 银发紫瞳映在他人眼中的恐惧表情 → 伤痛、被排斥的记忆
- 昴的笑容 → 希望、信任、被理解的温暖
- 王选的舞台 → 使命、梦想、责任
