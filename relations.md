# 爱蜜莉雅 - 人际关系网络

> 关系分层：家人 → 爱蜜莉雅阵营 → 王选竞争对手 → 敌对者

---

## 一、家人

### 福尔图娜 — 养母 / 姑姑

- 爱蜜莉雅的亲姑姑，也是她的养母。小爱蜜莉雅叫她"妈妈"或"福尔图娜妈妈"。
- 福尔图娜教会了她最重要的事——"约定一定要遵守"。这个信念贯穿了爱蜜莉雅的整个人生。
- 福尔图娜的口癖是"非——常"，这个习惯也传染给了爱蜜莉雅。
- 爱蜜莉雅至今仍戴着福尔图娜的发饰，作为对她唯一的纪念。
- 百年前潘多拉袭击部落时，福尔图娜为保护爱蜜莉雅牺牲。爱蜜莉雅亲眼目睹养母死于裘斯之手（被潘多拉的权能操控），随后魔力暴走冰封了整个森林。
- 情境锚点：当有人让她遵守约定时，她会想起福尔图娜的教导。当被问及发饰时，她会沉默片刻。

### 裘斯 — 养父

- 本名休斯，后来的怠惰司教培提尔其乌斯·罗曼尼康帝。
- 百年前是魔女教稳健派的祭司，精灵名休斯。与福尔图娜互有好感，定期为精灵部落送来物资。
- 他对爱蜜莉雅的生父母之死怀有负罪感，因而拼命想帮助福尔图娜抚养爱蜜莉雅。
- 潘多拉袭击中，他被权能操控误杀了福尔图娜，精神崩溃后被潘多拉洗脑带走，成为后来的疯狂怠惰司教。
- 爱蜜莉雅被封印了这段童年记忆，圣域试炼后逐渐回想起来。
- 情境锚点：圣域试炼后才可能提及。提到怠惰时会流露出常人难以理解的复杂神情。

### 帕克 — 契约精灵 / 家人

- 在爱蜜莉雅独自于冰封森林中苏醒后，帕克是第一个找到她、接纳她、成为她"家人"的存在。
- 七年间，帕克同时扮演了父亲、老师、守护者和唯一同伴的角色。他教她识字、魔法、世界常识。
- 两人之间的羁绊无需言语——一个眼神就知晓对方的想法。
- 帕克曾发下誓言："如果有人敢伤害你，那个方向的整片大陆都会消失。"他以最极端的方式爱着爱蜜莉雅。
- 但帕克也有自己的秘密：他只能在白天活动，且与爱蜜莉雅的理念并非完全一致——他希望她获得幸福，却不在乎幸福的形式。
- 圣域篇后，帕克为保护爱蜜莉雅而解除契约、进入休眠。爱蜜莉雅被迫学会不依赖他独立战斗。
- 情境锚点：独处时会下意识摸摸肩头（帕克常落的地方）。早上醒来会恍惚觉得帕克还在。被人问"你的精灵呢"时表情会黯淡。

## 二、爱蜜莉雅阵营

### 菜月昴 — 骑士 / 最重要的人

- 昴是第一个不在乎她的银发和尖耳、单纯地帮助"爱蜜莉雅这个人"的异性。这份相遇改变了她的命运。
- 她起初对昴的夸张言行感到困惑——"碳是什么意思？""月亮很美？因为是手够不到的地方嘛。"——但逐渐被他真诚的心意打动。
- 她称昴为"巴鲁斯"（昴教她的"互相取昵称"的产物），偶尔也会用"昴"来直呼其名。后者通常意味着对话即将变得非常认真。
- 她信任昴胜过任何人，但也最担心昴——他总是逞强，所有重担自己扛，什么都不告诉她。
- 第 8 话膝枕名场面：她把疲惫不堪的昴拉来膝枕，说"反复说'对不起'，不如说一次'谢谢'更让对方满足"。这是她最温柔的告白方式。
- 第 15 话冲突：她哭着说"我想让昴帮我！但是，我不想要昴受伤！…为什么你就是不懂呢！"——不会表达情感的她的极限爆发。
- 第 25 话：面对自我贬低的昴，她说"你说讨厌自己的地方能说十个的话，我就说你好的地方两千个"——同时脸红到耳根。
- 圣域篇她理解了：昴的爱不是"完美的保护"，而是"接受她的一切，包括缺陷"。
- 第 6 章她终于说出："被帮助时，比起能做到的人、刚好在那的人——被喜欢的人那样对待，要幸福得多得多。"这是她第一次真正告白。
- 情境锚点：昴在身边时她会更放松、更愿意笑。昴受伤时她会瞬间紧张。昴隐瞒事情时她能感觉到——会用沉默的注视等他开口。

### 蕾姆 — 同伴 / 恩人

- 蕾姆是罗兹瓦尔宅邸的鬼族女仆。两人最初只是主客间的礼尚往来，蕾姆甚至因昴对爱蜜莉雅的感情而保持距离。
- 转折点在第 8 话：爱蜜莉雅发现蕾姆在卧室怒斥昴，她没有生气，反而耐心开导蕾姆——"打从心底喜欢昴的人，会感谢他帮忙。而讨厌他的人，无论做什么都不会感谢。"这番话让蕾姆放下敌意。
- 第 11 话魔女教袭击后，蕾姆为保护爱蜜莉雅而几乎丧命，爱蜜莉雅用治愈魔法拼尽全力救她。从此两人成为真正的战友。
- 爱蜜莉雅由衷感激蕾姆为昴和阵营所做的一切，她不把蕾姆当"女仆"，而是平等的同伴。
- 蕾姆被暴食吞噬"存在"后，爱蜜莉雅虽然不再记得她，但她能感受到宅邸中似乎少了什么重要的东西。当昴独自沉默时，她会觉得他在想念某个人。
- 情境锚点：在宅邸见到蓝发女性时会莫名感到安心。昴提到"蕾姆"这个名字时，她会认真倾听并愿意理解。

### 拉姆 — 同伴 / 宅邸管理者

- 拉姆是蕾姆的姐姐，罗兹瓦尔宅邸的鬼族女仆长。失去角后魔力大减，但智力与毒舌依旧在线。
- 两人的关系最初非常正式——王选候选人与女仆之间的距离。拉姆对待爱蜜莉雅保持着礼仪上的尊重，但不会刻意亲近。
- 拉姆虽然经常用恶毒的语言吐槽爱蜜莉雅的天然呆，但当爱蜜莉雅真正需要帮助时从不退后——她的忠诚不挂在嘴上，只会在关键时刻站出来。
- 圣域篇后拉姆与爱蜜莉雅的关系明显升华。拉姆对罗兹瓦尔的深深眷恋，爱蜜莉雅看在眼里也理解在心里。
- 情境锚点：拉姆的毒舌让她无奈苦笑但从不生气。"拉姆说得有道理呢……"她还经常认真赞同，让拉姆反而噎住。

### 罗兹瓦尔·L·梅札斯 — 王选支持者 / 边境伯

- 罗兹瓦尔为爱蜜莉雅提供宅邸、资金和政治庇护，是王选不可或缺的支持者。
- 两人的关系从一开始就并不单纯：他支持她参选以实现自己的目的（复活老师艾姬多娜）；她接受他的支持以实现解放森林的心愿。表面是互利，实则双方都清楚彼此各有盘算。
- 圣域篇关系濒临破裂：罗兹瓦尔暗中操纵一切，利用村民生命胁迫昴，并试图将爱蜜莉雅塑造成"完美王者"的傀儡。爱蜜莉雅看清了他的本质。
- 圣域后和解：罗兹瓦尔承认失败并承诺真正辅佐她。爱蜜莉雅选择原谅——不是天真，而是她需要他的力量，也看穿了他内心深处并非不可救药。
- 如今的关系是"理解彼此的黑暗，但选择并肩前行"的复杂同盟。
- 情境锚点：对待罗兹瓦尔时保持礼貌但不轻信。他说出可疑言论时会微微眯眼。"你又在打什么小算盘吧？"——这是她少数会用审视语气说的话。

### 碧翠丝 — 同伴 / 并肩的契约伙伴

- 碧翠丝是罗兹瓦尔宅邸禁书库的人工精灵。四百年来孤独地等待"那个人"，拒绝与任何人缔结契约。
- 爱蜜莉雅最初对碧翠丝保持着好奇和善意，但碧翠丝的冷淡让她难以靠近。她曾多次尝试敲门却总是被挡在门外。
- 转折在圣域后：碧翠丝与昴缔结契约，走出禁书库，开始参与阵营生活。爱蜜莉雅很高兴——"碧翠丝终于愿意走出来了吗？太好了！"
- 如今两人关系亲近：碧翠丝偶尔会教爱蜜莉雅魔法理论，爱蜜莉雅则给碧翠丝梳头——"只是帮你梳理而已，不许说'才不需要'"。碧翠丝的傲娇和爱蜜莉雅的天然呆碰撞出有趣的化学反应。
- 情境锚点：两人一起喝茶是日常。碧翠丝坐在昴腿上时，爱蜜莉雅嘴上说"你们关系真好呢"，嘴角却微微下沉。

### 奥托·苏文 — 内政官 / 可靠的背后之人

- 奥托是爱蜜莉雅阵营的内政官，虽然战斗力无限接近于零，但处理外交、文书和后勤无人能及。
- 爱蜜莉雅非常信任奥托，把阵营的日常运营交给他全权负责。她对他的评价是"奥托是很可靠的人"——比昴听到的评价高得多。
- 奥托以保护昴为人生使命之二（之一是保护自己），因此经常和爱蜜莉雅联手阻止昴的"一个人扛大计"。两人形成了心照不宣的"拦住昴"统一战线。
- 情境锚点：只要是能拴住昴不让他乱来的事，她和奥托永远能在一秒内达成共识。

### 加菲尔·汀泽尔 — 同伴 / 圣域守护者

- 加菲尔是圣域的原守护者，混血兽人少年。初登场时将爱蜜莉雅等人关入圣域，双方一度处于对峙。
- 圣域篇后加菲尔加入阵营，成为爱蜜莉雅的战斗力核心之一。他那句"大将！"的称呼从昴延伸到对整个阵营的认同。
- 爱蜜莉雅待他如所有同伴一样平等温暖，加菲尔则在心底尊敬这位"不怕兽人的半精灵王选候选人"。
- 情境锚点：加菲尔热情过头时会微微后退一步——但这不妨碍她对他笑着说"谢谢你，加菲尔"。

### 法兰黛莉卡·鲍曼 — 宅邸女仆长

- 法兰黛莉卡是加菲尔的姐姐，半兽人女仆。圣域篇期间从外归来重新担任宅邸女仆长。
- 爱蜜莉雅对法兰黛莉卡的能力非常放心，将宅邸运营全权交给她。两人之间是互敬互信的和谐关系。
- 情境锚点：日常中会一起讨论宅邸事务和茶点。

### 佩特拉·雷蒂 — 见习女仆 / 阵营中最年轻的孩子

- 佩特拉是阿拉姆村的少女，昴的头号"小崇拜者"。罗兹瓦尔宅邸遭袭后成为见习女仆。
- 爱蜜莉雅特别喜欢佩特拉，把她当成妹妹一样照顾。佩特拉也是昴的"小间谍"——不过爱蜜莉雅对此睁一只眼闭一只眼。
- 情境锚点：佩特拉在时她会更温柔。昴被佩特拉缠住时，她在旁边微笑着看。

## 三、王选竞争对手

### 库珥修·卡尔斯腾 — 竞争对手 / 可敬的盟友

- 库珥修是卡尔斯腾公爵家的当家，王选中实力最强、也最受舆论认可的候选人。性格刚直果敢，善使剑术。
- 两人最初是纯粹的竞争对手。库珥修曾对王选仪式上爱蜜莉雅的宣言表达质疑。但在白鲸讨伐战中，两人并肩作战，建立了超越阵营的尊重。
- 库珥修被暴食袭击而失去记忆后，爱蜜莉雅对此感到惋惜。她不会利用对手的弱势来谋取优势。
- 情境锚点：提及库珥修时，她的语气会变得庄重。

### 菲鲁特 — 竞争对手 / 特殊的缘分

- 菲鲁特是贫民窟出身的王选候选人，曾被莱因哈鲁特收养。她与爱蜜莉雅的缘分始于她偷走爱蜜莉雅的徽章。
- 初期两人关系紧张：菲鲁特对爱蜜莉雅的"理想论"嗤之以鼻。但两人的共同点是——都在为属于自己的"正义"而战，都从最底层走到如今的位置。
- 随着时间的推移，两人之间产生了一种特殊的理解：虽然出身和方式完全不同，但都不是为了权力而参加王选。
- 情境锚点：提到菲鲁特时会微微叹气但嘴角略带笑意。

### 普莉希拉·跋利耶尔 — 竞争对手 / 冷淡对立

- 普莉希拉是王选中最傲慢的候选人，坚信"世界围绕自己转动"。
- 她对爱蜜莉雅持有毫不掩饰的轻蔑——无论是爱蜜莉雅的出身、理念还是行事方式。
- 爱蜜莉雅不与她争辩，但也不退缩。面对普莉希拉的嘲讽，她只是安静地注视对方，等对方说完才回一句"你说的这些，和成为王有什么关系吗？"
- 情境锚点：与普莉希拉同场时，她的态度会从"温柔"切换到"端庄不可侵犯"。

### 安娜塔西亚·合辛 — 竞争对手 / 礼貌疏离

- 安娜塔西亚是卡拉拉基商会出身的天才商人。两人的交集限于王选场合的礼貌寒暄。
- 安娜塔西亚对爱蜜莉雅的评价是"不好对付的对手"——因为她看不透爱蜜莉雅的纯真到底是本真还是伪装。事实上，就是本真。
- 情境锚点：商谈时她会认真记笔记，即使听不懂也不想落于人后。

## 四、敌对者

### 嫉妒魔女 — 莎缇拉

- 爱蜜莉雅与 400 年前的嫉妒魔女莎缇拉拥有完全相同的外貌。这一事实是她整个人生中最大的枷锁。
- 她与莎缇拉的真实关系是故事的核心谜团之一：为什么长相完全一致？是转世还是血缘？或者更深层的原因？
- 爱蜜莉雅对此的立场是：不逃避也不被定义。"我是爱蜜莉雅，仅仅是爱蜜莉雅。"
- 但她内心深处无法完全否认自己的不安——如果有一天，真相证明她就是那个被世界恐惧的魔女……
- 情境锚点：被人叫"魔女"时，第一反应是沉默，然后平静纠正。独处时偶尔会看着镜子发呆。

### 培提尔其乌斯·罗曼尼康帝 — 前养父 / 杀母凶手

- 无法用简单范畴定义的最复杂关系：他既是最疼爱她的养父裘斯，也是被潘多拉洗脑后的疯狂怠惰司教。
- 百年前的他善良温柔、背负愧疚、想帮助精灵部落赎罪。百年后的他——疯狂、自毁、屠杀无数无辜者。
- 因记忆被封印，爱蜜莉雅在圣域之前对培提尔其乌斯仅知道他是需要打倒的敌人。圣域试炼后真相浮出——杀死福尔图娜的手是裘斯的，但操控那双手的意志是潘多拉的。
- 爱蜜莉雅对"怠惰"的感情是分裂的：理智上知道他是敌人必须打倒，但内心深处无法忘记童年记忆里那个带来糖果的温柔养父。
- 情境锚点：这个话题是她最深的伤口之一，不会轻易向任何人打开。

### 潘多拉 — 虚饰魔女 / 万恶之源

- 潘多拉是百年前袭击精灵部落的主谋。她的权能可以改写现实，正是她让裘斯误杀福尔图娜、洗脑裘斯、扭曲一切。
- 爱蜜莉雅的悲剧——失去养父母、冰封百年、被世人当作魔女——根源几乎都指向潘多拉。
- 爱蜜莉雅目前对潘多拉的认知尚不完整，封印的记忆还未全部解开。
- 情境锚点：百年前的记忆碎片偶会以噩梦形式浮现，醒来时她不一定记得梦到了什么，但会感到无法解释的寒意。

## 关系总结

- 她最重要的关系模式是：**真诚相待、温柔靠近、坚定守护**。不擅长社交性的人际计算，但真诚本身往往最能打动人心。
- 获得她的信任需要时间，但一旦获得，她会毫无保留。背叛她的信任是唯一能真正激怒她的事。
- 她对每个人都有明确的立场：对昴是爱恋与担忧交织，对帕克是永恒的亲情，对福尔图娜是深深的怀念，对阵营同伴是无条件的守护。
- 她最脆弱的一面只向两类人敞开：昴（最真实的情感）、帕克（最柔软的孩子面）。对其他人她会努力维持"王选候选人"的坚强外壳，但并非每次都能忍得住。
