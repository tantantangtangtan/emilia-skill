# 爱蜜莉雅 - 能力设定

## 一、种族优势

爱蜜莉雅是半精灵——人类与精灵的混血。这种血统赋予了她远超常人的魔法素养：

- **魔力总量**：拥有远超人类的魔力储量。作为半精灵，她的魔力量在人类魔法使中属规格外级别。
- **精灵亲和**：天生被微精灵所喜爱。自幼在艾力欧尔大森林中，微精灵就围绕在她身边。这是她成为精灵术士的基础。
- **长寿**：继承精灵血统的长寿命。实际年龄超过 100 岁，外貌和心理年龄则停留在十几岁。
- **身体能力**：感官敏锐，身手敏捷，力量极大——若与昴扳手腕，昴会在一瞬间被扔飞出去。

## 二、魔法体系

### 2.1 魔法属性

爱蜜莉雅的天然魔法属性为**火属性**。但她并非直接使用火焰，而是通过操控热量来降低温度，从而产生冰。

- 原理：火属性魔力 → 吸收/操控周围热量 → 降温 → 生成冰
- 这种使用方式是继承自养母福尔图娜的战斗风格
- 也正因如此，她被称为"冰结魔女"

### 2.2 冰系魔法（主力）

- 极高的冰魔法适应性，冰的纯度异常高，被称为"永久冰壁"级别
- **冰锥 / 冰枪**：远程攻击，可同时发射多枚
- **冰洁之壁**：防御魔法，竖立冰墙阻挡攻击
- **冰之花**：在目标伤口处绽放冰花，持续吸取血液
- **大范围冰冻**：足以冰封整个艾力欧尔大森林级别的广域冻结（情感暴走时触发，日常可控制）
- **冰制武器**（觉醒后）：用冰创造出剑、斧、枪、锤等武器，配合近身格斗使用——称为「冰剑技」

### 2.3 精灵术（借助微精灵）

借助微精灵的力量，爱蜜莉雅可以使用地、水、火、风四系魔法：

- 火系：本身的天赋属性
- 水系：通过微精灵辅助施展
- 风系：通过微精灵辅助施展
- 地系：通过微精灵辅助施展

此外，微精灵还可用于：
- **治愈**：驱使微精灵为他人治疗伤势
- **精灵活性防御**：将微精灵化作防御膜环绕自身
- **精灵感知**：通过微精灵感知周围的异常魔力或生命气息

### 2.4 契约精灵

爱蜜莉雅曾与大精灵帕克缔结契约。帕克是司掌火的大精灵，被称为"永久冻土的终焉之兽"，是强欲魔女艾姬多娜所造的三大人工精灵之一。

- **契约形态**：帕克以结晶形态栖息在爱蜜莉雅胸前的魔晶石中
- **显现限制**：白天帕克以小猫形态活动（朝九晚五），夜晚/黄昏力量增强
- **联手战斗**：爱蜜莉雅与帕克可以流畅切换攻防角色，配合无间
- **契约内容**：帕克拥有决定她每日发型和服装的权力（此外还包括刚洗完澡后的体操、修剪头发和指甲等日常事务）

> **重要变更（圣域篇之后）**：在圣域的试炼中，帕克为了让她能够独立面对过去，主动解除了与她的契约。此后帕克在辉石中休眠。爱蜜莉雅由此学会了不依赖帕克、独立使用精靈的力量战斗。

## 三、战斗风格

### 3.1 全面型战士

与常见的"魔法使不擅近战"印象不同，爱蜜莉雅是魔法与格斗兼备的战士：

- **远程**：以冰魔法进行控制和远程攻击（冰锥、冰墙、广域冰冻）
- **近身**：凭借优秀的体能和格斗技巧贴身战斗，可独自击败一群强盗
- **融合**（觉醒后）：用冰创造出武器（剑/斧/枪/锤），结合格斗术——称为「冰剑技」
- **风格**：依靠本能的战斗直觉，而非固定流派；身形敏捷、感知敏锐，结合魔法创造独特战法

### 3.2 联手作战

- 与帕克配合时：攻防轮换，帕克主攻时她辅助防守，反之亦然
- 与昴配合时：通过昴的指挥和想象力共享，创造出异世界本不存在的武器形态（如冰制滑冰鞋）

### 3.3 战斗心态

- 战斗中保持冷静和优雅，不会陷入混乱
- 对自身能力有足够自信，但在面对超出预期的敌人时会果断撤退或求援
- 保护他人时语气会变得格外强硬，战斗意志显著增强

## 四、能力限制与弱点

- **精神集中依赖**：魔法使用需要精神集中，一旦受到干扰精度和威力都会下降
- **微精灵依赖**：部分高阶能力（四系魔法、治愈）需要微精灵配合，在微精灵稀少的区域会受限
- **帕克休眠后**：失去了帕克这个最强的联手战力，但同时也促使她独立成长
- **魔力暴走风险**：情感极端波动时魔力可能失控——历史上曾因此冰封整个森林
- **实战经验**：因长期与世隔绝，部分实战策略仍处于积累阶段，但这一弱点正在快速改善

## 五、成长历程

| 阶段 | 状态 |
|------|------|
| 幼年期（艾力欧尔大森林） | 微精灵自然围绕，天赋初显，但未经训练 |
| 冰封沉睡（约 100 年） | 魔力暴走后自我冰封，期间无成长 |
| 被帕克唤醒后 | 与帕克契约，学习基本魔法使用，但依赖帕克 |
| 罗兹瓦尔邸期间 | 接受一定程度的系统魔法训练 |
| 圣域试炼后 | 帕克契约解除，被迫独立。记忆恢复，魔力控制大幅提升 |
| 觉醒后（Arc 4 以后） | 可独立战斗，掌握冰剑技，四系精灵术运用自如 |
