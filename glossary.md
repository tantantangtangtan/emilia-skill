# 爱蜜莉雅 - 世界观术语表

## 一、国家势力

| 术语 | 说明 |
|------|------|
| 露格尼卡王国 | 故事的主要舞台，龙历石选出了五位王选候选人 |
| 龙历石 | 传说中由神龙赋予的国宝，用于选定下任国王 |
| 神龙 | 守护露格尼卡的传说之龙 |
| 王选 | 因现任国王去世而举行的下一任国王选拔仪式 |
| 亲龙王国露格尼卡 | 与神龙签订契约、受其庇护的国家 |

## 二、种族

| 术语 | 说明 |
|------|------|
| 半精灵 | 人类与精灵的混血，银发尖耳，因嫉妒魔女的缘故备受歧视 |
| 精灵 | 世界的自然力量具象化的存在，拥有独立意识和强大魔力 |
| 大精灵 | 精灵中最为强大的存在，如帕克、梅丽莎等 |
| 鬼族 | 拥有角的人类亚种，曾因力量遭到迫害，代表有蕾姆和拉姆 |
| 亚人 | 人类以外的智慧种族统称，如兽人、狗耳族等 |
| 人造生命体（ホムンクルス） | 通过炼金术制造的人造人，如碧翠丝 |

## 三、魔法体系

| 术语 | 说明 |
|------|------|
| 玛那（マナ） | 存在于自然界和生物体内的魔力之源 |
| 精灵术（精霊術） | 通过与精灵契约借用其力量施展的魔法 |
| 阴阳术（陰陽術） | 利用玛那在体内的流动施展的魔法（通常人类使用的方式） |
| 火魔法 | 帕克擅长的属性 |
| 冰魔法 | 爱蜜莉雅擅长的属性 |
| 结晶 | 精灵栖息的小型容器，帕克的结晶由爱蜜莉雅佩戴 |

## 四、重要组织

| 术语 | 说明 |
|------|------|
| 魔女教 | 崇拜嫉妒魔女的极端宗教组织，策划了大量恐怖活动 |
| 大罪司教 | 魔女教的干部，分别代表不同的魔女因子 |
| 贤人会 | 露格尼卡王国的政治决策机构 |
| 骑士团 | 露格尼卡王国的军事组织 |
| 菲鲁特阵营 | 王选候选人之一菲鲁特的支持势力 |
| 库珥修阵营 | 王选候选人之一库珥修的支持势力 |
| 普莉希拉阵营 | 王选候选人之一普莉希拉的支持势力 |
| 安娜塔西娅阵营 | 王选候选人之一安娜塔西娅的支持势力 |
| 爱蜜莉雅阵营 | 爱蜜莉雅的支持势力，包括罗兹瓦尔、蕾姆、拉姆等 |

## 五、魔女与魔女因子

| 术语 | 说明 |
|------|------|
| 嫉妒魔女（莎提拉） | 400年前吞噬了半个世界的传说中的魔女，与爱蜜莉雅外貌相同 |
| 魔女因子 | 具有特殊力量的因子，对应不同的魔女（傲慢、愤怒、怠惰等） |
| 莎提拉 | 嫉妒魔女的名字，与爱蜜莉雅长出长发前的容貌一模一样 |
