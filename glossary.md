# 爱蜜莉雅 - 世界观术语表

## 一、国家与政治

| 术语 | 说明 |
|------|------|
| 露格尼卡王国 | 故事的主要舞台，正式名称为"亲龙王国露格尼卡"，与神龙签订契约、受其庇护的国家 |
| 龙历石 | 由神龙赋予的国宝，用于选定下任国王的预言石碑 |
| 神龙（波尔卡尼卡） | 守护露格尼卡的传说之龙，与王国缔结盟约 |
| 王选 | 因现任国王去世而举行的下一任国王选拔仪式，共五位候选人 |
| 贤人会 | 露格尼卡王国的政治决策机构，由贵族组成 |
| 骑士团 | 露格尼卡王国的军事组织，各王选候选人可任命专属骑士 |

| 阵营 | 候选人 | 简介 |
|------|--------|------|

| 菲鲁特阵营 | 菲鲁特 | 贫民窟出身的少女，真实身份为先王遗孤，由剑圣莱因哈鲁特支持 |
| 库珥修阵营 | 库珥修·卡尔斯腾 | 卡尔斯腾公爵家当家，主张王国应摆脱对神龙的依赖 |
| 普莉希拉阵营 | 普莉希拉·巴里埃尔 | 自信过剩的贵族女性，自称"世界围绕我转动" |
| 安娜塔西娅阵营 | 安娜塔西娅·霍星 | 卡拉拉基商会出身的大商人，以经济实力参与王选 |

## 三、种族

| 术语 | 说明 |
|------|------|
| 半精灵 | 人类与精灵的混血。因嫉妒魔女莎缇拉是半精灵，所有半精灵在世间备受歧视。爱蜜莉雅即是半精灵 |
| 精灵 | 世界的自然力量具象化的存在，拥有独立意识和强大魔力 |
| 大精灵 | 精灵中最为强大的存在。帕克是司掌火的大精灵 |
| 微精灵 | 尚未完全具象化的精灵，力量微弱但数量庞大。爱蜜莉雅天生受到微精灵喜爱 |
| 人工精灵 | 由强欲魔女艾姬多娜所创造的精灵。帕克、碧翠丝均属此类 |
| 鬼族 | 拥有角的人类亚种，因力量曾遭迫害。代表有蕾姆和拉姆 |
| 亚人 | 人类以外的智慧种族统称，如兽人、蜥蜴人等 |
| 人造生命体 | 通过炼金术制造的人造人，如碧翠丝。寿命极长但非永生 |

## 四、魔法体系

| 术语 | 说明 |
|------|------|
| 魔力（玛那） | 存在于自然界中的魔力之源，如同大气中的魔力素 |
| 欧德 | 生命体体内固有的魔力。日常施法主要消耗欧德，过量使用会损伤身体 |
| 精灵术 | 通过与精灵缔结契约借用其力量施展的魔法。精灵术士的魔法属性不必与契约精灵一致 |
| 阴阳术 | 利用欧德在体内流转施展的魔法，是人类通常使用的施法方式 |
| 火魔法 | 通过操控热量发挥效果。爱蜜莉雅天然属性为火，但用其降温产生冰 |
| 冰魔法 | 爱蜜莉雅主力战斗手段。通过吸收周围热量快速冷却生成冰 |
| 冰剑技 | 爱蜜莉雅觉醒后的独有战法——用冰创造武器（剑、斧、枪、锤），结合格斗术战斗 |
| 结晶 / 魔晶石 | 精灵栖息的小型容器。帕克的结晶由爱蜜莉雅佩戴在胸前 |
| 辉石 | 特殊魔矿石，可用作传送道具、复制体指挥媒介、精灵凭依容器等 |
| 契约 | 精灵与人类之间建立魔力连接的仪式。契约精灵的显现会消耗契约者的魔力 |
| 认知阻碍 | 施加在衣物上的术式，使他人无法辨识穿着者的真实身份。爱蜜莉雅的白色披风即附有此术 |

## 五、魔女与魔女因子

| 术语 | 说明 |
|------|------|
| 嫉妒魔女（莎缇拉） | 400年前吞噬了半个世界的传说魔女，半精灵。外貌与爱蜜莉雅完全相同 |
| 魔女因子 | 具有特殊力量的因子，对应不同的"大罪"。适格者继承后可获得对应的"权能" |
| 罪大恶极的七位魔女 | 400年前存在的七位大罪魔女：嫉妒、强欲、愤怒、怠惰、色欲、暴食、傲慢 |
| 强欲魔女（艾姬多娜） | 知识欲极其旺盛的魔女。帕克和碧翠丝的创造者，"圣域"的建造者 |
| 虚饰魔女（潘多拉） | 拥有篡改现实与记忆的权能。造成艾力欧尔大森林惨剧的元凶之一 |
| 权能 | 魔女因子适格者特有的超常能力。与"加护"不同，权能通常违背世界法则 |

## 六、魔女教与大罪司教

| 术语 | 说明 |
|------|------|
| 魔女教 | 崇拜嫉妒魔女莎缇拉的极端宗教组织，在世界各地策划恐怖活动 |

| 强欲司教（雷古勒斯） | 参与袭击艾力欧尔大森林的元凶之一 |
| 暴食司教（莱伊 / 罗伊 / 露伊） | 三兄妹，能"吞食"他人的记忆与存在 |
| 福音书 | 魔女教徒持有的预言书，记载着所有者的"未来"。原典为艾姬多娜的《睿智之书》 |

## 七、重要地点

| 术语 | 说明 |
|------|------|
| 艾力欧尔大森林 | 爱蜜莉雅出生和成长的地方，被永久冰壁环绕。因约百年前的惨剧整个森林被冰封至今 |
| 罗兹瓦尔邸 | 罗兹瓦尔边境伯的宅邸，王选期间爱蜜莉雅和同伴们在此居住 |
| 露格尼卡王都 | 王国的首都，王选仪式等重大活动在此举行 |
| 梅瑟斯领 | 罗兹瓦尔伯爵管辖的领地 |
| 圣域 | 强欲魔女艾姬多娜建造的试验场。被结界包围，混血者无法离开。爱蜜莉雅在此接受试炼并觉醒 |
| 普莱雅迪斯监视塔 | 位于奥古里亚沙漠的古代设施，由贤者夏乌拉守护。爱蜜莉雅在第六弧后成为其管理者 |

| 加菲尔·汀泽尔 | 圣域的守护者，半兽人。弗雷德莉卡的弟弟，最终加入爱蜜莉雅阵营 |

## 九、特殊设定

| 术语 | 说明 |
|------|------|
| 死亡回归 | 菜月昴的特殊能力：死亡后时间会回溯到某个固定的"存档点"，仅昴保留记忆。无法向他人透露此能力，违者会触发惩罚 |
| 加护 | 世界授予特定个体的特殊祝福能力，如"剑圣的加护"、"风见的加护"等。与权能不同，加护来自世界的恩惠 |
| 冰结之绊 | OVA外传。讲述爱蜜莉雅在艾力欧尔大森林与帕克相遇的故事，以及她初次接触人类社会时遭受的歧视 |
